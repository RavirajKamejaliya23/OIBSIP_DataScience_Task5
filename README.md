# OIBSIP_DataScience_Task5

## Sales Prediction using Machine Learning

### Objective

The objective of this project is to build a machine learning model that predicts product sales based on advertising expenditures across different media channels such as TV, Radio, and Newspaper.

### Dataset

The dataset contains:

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget
* Sales

### Tools and Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

### Steps Performed

1. Loaded and explored the dataset.
2. Performed data cleaning and preprocessing.
3. Conducted exploratory data analysis (EDA).
4. Visualized relationships between advertising budgets and sales.
5. Built a Linear Regression model.
6. Evaluated model performance using R² Score, MAE, and MSE.
7. Predicted future sales based on advertising expenditure.

### Result

Successfully developed a machine learning model capable of predicting sales based on advertising investments with good accuracy.

### Outcome

This project enhanced understanding of regression analysis, predictive modeling, data visualization, and the practical application of machine learning in business decision-making.
